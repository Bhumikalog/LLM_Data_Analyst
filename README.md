# LLM Data Analyst

An AI-powered data analysis assistant that converts natural-language queries into executable Pandas workflows using Groq LLMs.

## Features

- Natural Language → Python Code Generation
- Automated Dataset Profiling
- Data Cleaning Pipeline
- Safe Code Execution
- Dynamic Visualizations
- LLM-generated Explanations

## Tech Stack

- Python
- Pandas
- Groq API
- Matplotlib
- Seaborn

## Example Queries

"What category has the highest total sales?"

"Show top 10 products by revenue"

"Plot sales by region"

## Architecture

User Query
↓
LLM Prompt Generation
↓
Code Generation
↓
Safe Execution
↓
Visualization & Insights
